# **Welcome to HOSS**

***

HOSS is a highly available, scalable, manageable,distributed object storage system based on HDFS. 

# **Build HOSS**

***
```
cd hoss 
ant clean & ant
```
# **Run HOSS**

***
```
start-hoss.sh 
stop-hoss.sh 
```
